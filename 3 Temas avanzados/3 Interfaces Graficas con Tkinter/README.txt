Se usara Tkinter.

Tiene varios componentes:
TK : raiz de todo.
Frame: Debe estar dentro de TK y puede contener otros frames ademas de los widgets.
Label, Button, Text, Entry: tipos de widget que estan contenidos en frames. 

si se guarda con extension pyw los archivos py, no saldra la terminal por debajo.
(en los archivos con extension py, al abrirlos directamente con python se abre un terminal debajo)
